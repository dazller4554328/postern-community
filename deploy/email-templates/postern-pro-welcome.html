You install Postern on your own VPS or home server — there is no Postern cloud. The bootstrap below does everything in one command.

1. Pick a host

Ubuntu 22.04 / 24.04 LTS or Debian 12, 2 vCPU, 4 GB RAM (or 2 GB + swap), 20 GB disk. A Hetzner CX22 (~€4/month) is plenty. Full requirements: docs.postern.email/install.

2. Install Tailscale

This is how you'll reach Postern from your phone — no public DNS or port-forwarding. SSH into your host and run:

curl -fsSL https://tailscale.com/install.sh | sh
sudo tailscale up

Open the URL it prints, sign in to your free Tailscale account, then generate an auth key at admin/settings/keys (tick Reusable + Ephemeral). Keep that tskey-auth-... value handy for the next step.

3. Install Postern

Still in the host's SSH session:

curl -fsSL https://updates.postern.email/install.sh \
  | sudo LICENSE={$service_username} bash

The bootstrap validates your license, downloads the signed Pro tarball (verifies the Ed25519 signature before extracting), then prompts for the Tailscale auth key. Total time: ~10 minutes including the first build.

When it's done, the script prints a URL like https://postern.<your-tailnet>.ts.net. Open it in any browser on a device that's also on your tailnet (phone, laptop) to reach Postern's first-boot screen.

4. First-boot setup

  1. Set a master password. This derives the encryption key for the SQLCipher vault — Postern never stores it. If you forget it, your mail is unrecoverable. Put it in your password manager before you click continue.
  2. Add your first mailbox. The wizard prefills IMAP/SMTP for Gmail, Fastmail, iCloud, and ProtonMail (via Bridge). Gmail needs an app password — the regular account password won't work for IMAP.
  3. Activate your license. Go to Settings → Updates; the key is already pre-filled, just click Activate.

Initial IMAP sync runs in the background. Empty mailbox: seconds. A decade of Gmail: an hour or two. Watch the unread badge tick up.
